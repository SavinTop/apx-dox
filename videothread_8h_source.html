33/*
34 * urisourcebin -> parsebin -> tee -> decodebin -> videoconvert -> tee -> appsink
35 * │ │
36 * ┌───────────────────────────┘ │
37 * │ │
38 * queue -> matroskamux -> filesink (without reencoding) │
39 * │
40 * ┌───────────────────────────────────────────────────────────────┘
41 * │
42 * queue (probe, draw overlay) -> videoconvert -> x264enc -> matroskamux -> filesink
43 *
44 */
45
46/* URI examples:
47 * avf://index0 - macos webcam device 0
48 * tcp://192.168.1.10:8765 - mpegts h264 stream
49 * udp://0.0.0.0:8765?codec=h264 - rtp stream with h264(h265)
50 * any other uri that supported by gstreamer
51 */
